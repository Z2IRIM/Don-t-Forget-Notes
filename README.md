# Don't Forget Notes

给Windows用的桌面便签应用，用React + Electron

## 已实现功能

- 便签列表：新增、删除、复制、搜索、置顶
- 富文本编辑：粗体、斜体、下划线、标题、列表、任务列表、引用、代码块、链接
- 文本样式：字体、字号
- Markdown 支持：
  - 可查看/编辑 Markdown 原文
- 桌面便签窗口：
  - 可生成多个独立桌面便签
  - 可拖动（顶部拖拽条）
  - 可拉动边缘调整大小
  - 可冻结（鼠标穿透，不响应悬停和点击）
- 外观系统：
  - 目前仅支持暗色
  - 玻璃标签 / 彩色标签 / 纸张标签 / 极简标签（还未开发完成）
  - 自定义强调色、透明度
- 系统集成：
  - 托盘菜单
  - 单例模式
  - 开机自启（打包后可用）
  - 静默启动（打包后可用）
  - 隐藏任务栏图标选项
