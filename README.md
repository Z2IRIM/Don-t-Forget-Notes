# Material Notes Desktop

一个面向 Windows 11 的桌面便签应用，采用 **React + Electron + Material Design**。

## 已实现功能

- 便签列表：新增、删除、复制、搜索、置顶
- 富文本编辑：粗体、斜体、下划线、标题、列表、任务列表、引用、代码块、链接
- 文本样式：字体、字号、字体颜色
- Markdown 支持：
  - Tiptap Markdown 扩展双向转换
  - 可查看/编辑 Markdown 原文
  - 可导出 `.md`
- 桌面便签窗口：
  - 可生成多个独立桌面便签
  - 可拖动（顶部拖拽条）
  - 可拉动边缘调整大小
  - 可冻结（鼠标穿透，不响应悬停和点击）
- 外观系统：
  - 明暗模式 / 跟随系统
  - 玻璃标签 / 彩色标签 / 纸张标签 / 极简标签
  - 自定义强调色、透明度、总在最前
- 系统集成：
  - 托盘菜单
  - 单例模式
  - 开机自启（打包后可用）
  - 静默启动（打包后可用）
  - 隐藏任务栏图标选项

## 技术栈

## 使用说明

### 桌面便签拖动
未冻结时，便签顶部标题栏是拖动区域。

### 桌面便签缩放
未冻结时，便签四边和四角有隐藏的 resize handle，可以直接拖动边缘改变大小。

### 冻结 / 解冻
冻结后，窗口启用鼠标穿透。

解冻方法：

- 打开主窗口，关闭冻结
- 托盘菜单里点击“解冻全部便签”

## 说明

1. 该版本把 **HTML 作为展示真值**，Markdown 作为导出/导入辅助格式，因此复杂字体样式以桌面显示效果为准。
2. 开机自启动和静默启动在开发模式下没有实际意义，建议打包安装后再测试。
3. 如果你想进一步增强到生产级，可以继续增加：

   - 历史版本

   - 全局快捷键
   - 真正的“贴桌面层”实现
